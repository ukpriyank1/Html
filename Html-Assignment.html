Q-5. What are different types of lists in HTML?

Lists are used to put together relevant pieces of information so that they are easily readable and clearly related to one another. Lists are essential components in contemporary web development and are regularly used for both navigation and general content.

The three list types

  • unordered list

    • -used to arrange a collection of connected elements without regard to order.

  • ordered list

    • - used to show name/value pairs, such as words and definitions, in a

  • description list

    • - certain order and to group a set of related objects.

Unordered list

When a group of elements can be arranged in any order, unordered (bulleted) lists are utilized. A shopping list is an illustration:

  • milk
  • bread
  • butter
  • coffee beans

Unordered list Markup

Unordered lists are composed of one set of <ul> </ul> tags followed by one or more sets of <li> </li> tags:

<ul>
   <li> bread </li>
   <li> cofee beans </li>
    <li> milk </li>
   <li> butter </li>
</ul>

Ordered lists

In order to present a list of things that must be in a particular sequence, ordered (numbered) lists are utilized. An illustration would be a recipe:

  • 1.assemble the ingredients
  • 2. Combine ingredients.
  • 3. Ingredients should be placed in a baking dish and baked for one hour.
  • 4. Take the oven out.
  • 5. 10 minutes should be given for standing.
  • 6. Serve

Ordered lists can be displayed in a variety of ways. Decimal numbers are typically the default in browsers, however there are additional options as well:

  • Letters
    • Lowercase ascii letters (a, b, c…)
    • Uppercase ascii letters (A, B, C…).
  • Numbers
    • Decimal numbers (1, 2, 3…)
    • Roman numerals (i, ii, iii…)

Ordered list markup

Ordered lists use one set of <ul> </ul> tags wrapped around one or more sets of <li> </li> tags:

<ul>
   <li> assemble the ingredients </li>
   <li>Combine ingredients. </li>
    <li> Ingredients should be placed in a baking dish and baked for one hour. </li>
   <li> Take the oven out. </li>
   <li> 10 minutes should be given for standing. </li>
   <li> Serve </li>
</ul>

Description lists

Definition lists, which were renamed to description lists in HTML5, are lists that link particular names and values. Examples include the elements in an ingredient list and their brief descriptions, the writers of articles, and the years that they won competitions. There must be at least one name and at least one value in each pair regardless of how many name-value groupings you have.

Since they are versatile, description lists allow you to link several values to a single name or vice versa.
You may demonstrate each meaning of the word "coffee," for instance, one after the other:

coffee
a coffee-based beverage that has been ground and roasted
an iced coffee
a social event where medium- to dark-brown coffee is drank

Description list markup

One or more groups of <dt> </dt> (name) and <dd> </dd> (value) tags are wrapped around one or more sets of <dl> </dl> (description list) tags. It is required to match at least one dt with at least one dd, with the dt always coming first in the source sequence.

A simple description list of single names with single values would look like this:

<dl>
   <dt> Name </dt>
   <dd>Value </dd>
    <dt> Name </dt>
   <dd> Value </dd>
</dl>

Q.9 How is Cell Padding different from Cell Spacing?

Cellpadding :

The cellpadding attribute in HTML table formatting is used to specify the amount of space between a cell's edge and its content. Simply said, cellpadding is an attribute of the table tag (<table>) that determines how far the cell border and content are separated from one another.
The cellpadding property in HTML accepts values in pixels or percentages. Depending on the needs of the user, this value could be 0. The table tag's cellpadding attribute is supported in older versions of HTML, however it is not supported in HTML5.

Syntax

<table cellpadding="value" >.....</table>

Cellspacing :

In HTML, cell spacing is yet another attribute of the table tag. It regulates the separation between two neighboring cells. As a result, a site developer can quickly alter the distance between consecutive cells' edges. It is primarily used to make the table more readable.

Syntax

<table cellspacing="value" >.....</table>

Q-18. What is SVG?

Scalable Vector Graphics, the abbreviation for the HTML SVG, is a type of graphic.

A modular language called HTML SVG is used to describe visuals in XML. It describes mixed vector/raster and two-dimensional vector graphics in XML. The W3C has recommended it. XML text files contain the definitions of SVG images and their behaviour. An SVG image can therefore be created and edited as XML files, but generally speaking, drawing programs like inkspace are preferred to generate it.

SVG is mostly used for vector diagrams, such as pie charts and 2-D graphs with an X, Y coordinate system.

The root of an SVG fragment is specified by the svg> element. Every component and attribute in SVG files can be animated.
